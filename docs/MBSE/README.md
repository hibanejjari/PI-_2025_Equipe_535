##  MBSE Analysis : VINCI Energies Dashboard Supervision Platform

 (Model-Based Systems Engineering )
 
### 1. Environment Analysis
Defines the system context, external stakeholders, and constraints.

**System of Interest:** Dashboard Supervision Platform (Python + dbt + PostgreSQL + Superset)  
**Stakeholders:** VINCI Engineers, Project Managers, Data Engineers, CI/CD Systems  
**Constraints:** Data governance, security, performance requirements

### 2. Functional Breakdown Structure (FBS)
Describes the system’s main functions using verbs only.

**Main functions:**
- Ingest data  
- Transform and process data  
- Monitor dashboards and ensure quality  
- Automate workflows  
- Provide user interfaces

### 3. Technical Interactions
Shows how system components and external tools interact.

**Layers:**
- **Data Management Layer:** Ingest, transform, clean  
- **Monitoring & Analytics Layer:** Compute KPIs, ensure compliance  
- **Automation & Visualization Layer:** Trigger actions, refresh dashboards
